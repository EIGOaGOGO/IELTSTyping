🔥 Firebase Setup
Connect a free Firebase database so all students share one live leaderboard.
This takes about 5 minutes and only needs to be done once.
1Create a Firebase project

Go to console.firebase.google.com → click Add project → give it a name → disable Google Analytics (optional) → click Create project.

2Add a Web App

Inside your project, click the </> icon (Web) → give the app a nickname → click Register app. You will see a firebaseConfig object — copy it all.

3Enable Firestore

In the left sidebar go to Build → Firestore Database → click Create database → choose Start in test mode → select any region → click Enable.
